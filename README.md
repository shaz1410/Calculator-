#Python Calculator (CLI)

This is a simple **Command-Line Calculator** built in Python.  
It allows users to perform basic arithmetic operations through a menu system.

---

##Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Power (**)
- Modulus (%)
- Input validation (prevents crashing on wrong input)
- Handles division by zero
- Runs continuously until the user exits

---

##  Technologies Used

- Python 3
